# مكتب الغانم للبرمجة — Ghanem Mobile Landing Page

صفحة هبوط احترافية متجاوبة (موبايل / تابلت / ديسكتوب) مبنية بـ HTML + CSS + JavaScript فقط — بدون أي مكتبات خارجية، سريعة التحميل وجاهزة للنشر مباشرة.

## بنية المشروع

```
obiade/
├── index.html          ← الصفحة الرئيسية (كل الأقسام)
├── css/
│   └── style.css       ← التصميم كامل (الألوان، الأنيميشن، التجاوب)
├── js/
│   └── main.js         ← واتساب، القائمة، الأنيميشن، العدّادات
├── assets/
│   └── img/            ← صورتان فقط تُستخدمان في الموقع
│       ├── logo.jpeg   ← الشعار الدائري (الهيرو، من نحن، التواصل، الفوتر)
│       └── banner.jpeg ← البانر العريض (نهاية قسم "كيف نعمل")
└── image/              ← الصور الأصلية المرجعية (غير معروضة في الموقع)
```

## التشغيل

افتح `index.html` مباشرة في المتصفح، أو شغّل سيرفر محلي:

```bash
npx serve .
```

## التخصيص — أين تغيّر كل شيء

### 1. رقم الواتساب (أهم شيء)

في أول ملف `js/main.js` — قيمة واحدة تتحكم بكل أزرار الواتساب في الصفحة:

```javascript
const WHATSAPP_NUMBER = '972597774407'; // بدون + وبدون مسافات
```

### 2. الرسائل المعبأة مسبقاً

كل زر واتساب يحمل رسالته الخاصة في `index.html` عبر الخاصية `data-wa-message`:

```html
<a href="#" data-wa-message="مرحباً، أريد إنشاء وتفعيل محفظة جوال باي.">...</a>
```

غيّر نص الرسالة كما تريد — سيتم ترميزها تلقائياً.

### 3. رقم الاتصال المباشر

ابحث في `index.html` عن `tel:+972597774407` وغيّره (موجود في قسم التواصل والزر السفلي).

### 4. الصور

الموقع يعرض صورتين فقط: `assets/img/logo.jpeg` و `assets/img/banner.jpeg`.
استبدلهما بنفس الأسماء لتحديثهما — باقي الأقسام مصممة بأيقونات وعناصر UI بدون صور.

### 5. نصوص الخدمات والأسعار

كل خدمة عبارة عن بطاقة `<article class="service-card">` في قسم `#services` — عدّل الاسم والوصف، ويمكنك إضافة سطر سعر داخل البطاقة إن أردت:

```html
<span class="service-price">50 ₪</span>
```

### 6. آراء العملاء

بطاقات `blockquote.review-card` في قسم `#reviews` — استبدل النصوص والأسماء بآراء حقيقية.

### 7. روابط السوشيال ميديا

في قسم التواصل، ابحث عن `class="socials"` واستبدل `href="#"` بروابطك الفعلية (فيسبوك / إنستغرام / تيليجرام).

### 8. ساعات العمل والعنوان

في قسم `#contact` — بطاقات "العنوان" و"ساعات العمل".

### 9. ألوان الهوية

كل الألوان في أعلى `css/style.css` ضمن `:root` — غيّر قيمة واحدة وستنعكس على الموقع كله:

```css
--green-400: #74d13d;  /* الأخضر الأساسي */
--navy-900: #07131f;   /* الخلفية الداكنة */
```

## المزايا التقنية

- تصميم مستخرج من هوية المحل (أخضر جوال باي + كحلي تقني داكن + لمسات زرقاء).
- دعم كامل للعربية RTL وخطوط Cairo / Tajawal.
- زر واتساب عائم + أزرار واتساب برسائل مخصصة في الهيرو والخدمات والتواصل.
- أنيميشن ظهور عند التمرير + عدّادات متحركة + تأثيرات hover (تحترم `prefers-reduced-motion`).
- SEO: عناوين منظمة، وصف meta، وسوم Open Graph، وبيانات LocalBusiness منظمة (Schema.org).
- تحميل كسول للصور (`loading="lazy"`) لسرعة أعلى.
